# HVAC 预测程序 改进方案

## 📋 需求总结

根据"所需数据"文档，需要在现有程序基础上实现三个步骤：

1. **步骤1：模型构建与比较**
   - 实现 RF (Random Forest) / XGBoost / DNN+LSTM 三模型对比
   - 输出 MAPE, R², RMSE 三个指标的表格
   - 选出精度最高的三个模型
   - 针对二次网五个分区供回水温度进行预测对比

2. **步骤2：模型优化（PSO 超参数优化）**
   - 使用 PSO (Particle Swarm Optimization) 优化 DNN+LSTM 的超参数
   - 10折交叉验证框架
   - 以 MAPE 为优化目标
   - 输出优化后的超参数表、预测结果、残差直方图

3. **步骤3：SHAP 可解释性分析**
   - 计算每个特征的 SHAP 值分布
   - 计算平均绝对 SHAP 值作为特征重要性指标
   - 排序所有特征变量的 SHAP 重要性

```python
import numpy as np
from sklearn.model_selection import KFold
from pyswarms.single.global_best import GlobalBestPSO
import tensorflow as tf
from sklearn.metrics import mean_absolute_percentage_error

class PSO_DNNLSTM_Optimizer:
    def __init__(self, X_train, y_train, n_splits=10, n_particles=20, n_iterations=12):
        """
        Args:
            X_train, y_train: 训练数据
            n_splits: K折数（默认10）
            n_particles: PSO粒子数（默认20）
            n_iterations: PSO迭代次数（默认12）
        """
        self.X_train = X_train
        self.y_train = y_train
        self.kfold = KFold(n_splits=n_splits, shuffle=True, random_state=42)
        self.best_params = None
        self.best_score = float('inf')
        
    def objective_function(self, params):
        """
        PSO 目标函数：最小化交叉验证 MAPE
        
        params: [lstm_units, dense_units, dropout, learning_rate]
        """
        lstm_units = int(params[0] * 256)  # 32-256
        dense_units = int(params[1] * 256)  # 32-256
        dropout = params[2] * 0.5  # 0-0.5
        lr = 10 ** (-params[3] * 4 - 5)  # 1e-5 to 1e-9
        
        cv_mape_scores = []
        for train_idx, val_idx in self.kfold.split(self.X_train):
            X_fold_train, X_fold_val = self.X_train[train_idx], self.X_train[val_idx]
            y_fold_train, y_fold_val = self.y_train[train_idx], self.y_train[val_idx]
            
            # 构建临时模型（简化版，可用前面的 build_optimized_model）
            model = self._build_temp_model(lstm_units, dense_units, dropout, lr, X_fold_train.shape)
            
            # 训练（少轮次加速）
            model.fit(X_fold_train, y_fold_train, epochs=5, batch_size=32, verbose=0)
            
            # 评估
            y_pred_val = model.predict(X_fold_val, verbose=0)
            mape = mean_absolute_percentage_error(y_fold_val, y_pred_val)
            cv_mape_scores.append(mape)
            tf.keras.backend.clear_session()
        
        mean_mape = np.mean(cv_mape_scores)
        return mean_mape
    
    def optimize(self):
        """执行 PSO 优化"""
        options = {'c1': 0.5, 'c2': 0.3, 'w': 0.9}
        optimizer = GlobalBestPSO(n_particles=20, dimensions=4, options=options, init_pos=None)
        cost, pos = optimizer.optimize(self.objective_function, iters=12, verbose=True)
        
        # 解析最优参数
        self.best_score = cost
        self.best_params = {
            'lstm_units': int(pos[0] * 256),
            'dense_units': int(pos[1] * 256),
            'dropout': pos[2] * 0.5,
            'learning_rate': 10 ** (-pos[3] * 4 - 5)
        }
        return self.best_params, self.best_score
```

```python
import shap

def compute_shap_importance(model, X_sample, feature_names, n_samples=100):
    """
    计算 SHAP 特征重要性
    
    Args:
        model: 训练好的模型（支持 sklearn/tf）
        X_sample: 样本数据
        feature_names: 特征名称列表
        n_samples: 用于 SHAP 计算的样本数
    
    Returns:
        shap_df: 包含特征排名的 DataFrame
    """
    # 如果是深度学习模型
    if hasattr(model, 'predict'):
        explainer = shap.DeepExplainer(model, X_sample[:n_samples])
        shap_values = explainer.shap_values(X_sample[:n_samples])
        
        # 计算平均绝对 SHAP 值
        if isinstance(shap_values, list):
            # 多输出情况：取平均
            mean_abs_shap = np.mean([np.abs(sv).mean(axis=0) for sv in shap_values], axis=0)
        else:
            mean_abs_shap = np.abs(shap_values).mean(axis=0)
    else:
        # sklearn 模型
        explainer = shap.TreeExplainer(model)
        shap_values = explainer.shap_values(X_sample[:n_samples])
        mean_abs_shap = np.abs(shap_values).mean(axis=0)
    
    # 构建 DataFrame
    shap_df = pd.DataFrame({
        '特征': feature_names,
        '平均绝对SHAP值': mean_abs_shap,
        '重要性排名': range(1, len(feature_names) + 1)
    }).sort_values('平均绝对SHAP值', ascending=False).reset_index(drop=True)
    shap_df['重要性排名'] = range(1, len(shap_df) + 1)
    
    return shap_df

def plot_shap_importance(shap_df, save_path='results/shap_feature_importance.png'):
    """绘制 SHAP 重要性条形图"""
    plt.figure(figsize=(10, 6))
    plt.barh(shap_df['特征'], shap_df['平均绝对SHAP值'])
    plt.xlabel('平均绝对 SHAP 值')
    plt.title('特征重要性排名（SHAP）')
    plt.tight_layout()
    plt.savefig(save_path, dpi=300)
    plt.close()
```

## ⚠️ 关键注意事项

1. **数据量要求**
   - PSO 优化需要充足数据（建议 >2000 样本）
   - SHAP 分析建议用测试集的代表性子集

2. **计算资源**
   - PSO 优化耗时较长（预计 30-60 分钟）
   - 建议在有 GPU 的环境下运行

3. **超参数选择**
   - PSO 粒子数 20、迭代次数 12 为默认配置
   - 可根据计算能力调整（少则快但精度低，多则慢但精度高）

4. **向后兼容**
   - 新增代码应不影响现有的 RF/XGB/DL 训练流程
   - 建议在 `main()` 中用条件判断控制各阶段执行
